get /games

get all the games

  • Response

HTTP status code 200

Body

Media Type: application/json

Type:

  • id: required (number)
  • pieces: required (Piece[])
  • locked: required (boolean)

    if true, game setup cannot be changed

    • outcome

    once game is finished, the results of that game
    Properties
    • created: required (datetime)
    • winner: required (string)

      team name of winner, or null if a draw

    • team_stats: required (TeamStats[])

      if played, all the stats for that game

    • pieces: required (Piece[])
